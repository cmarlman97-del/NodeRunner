# Brand Assets

Place static brand files here. These are served from `/brand/<filename>` at runtime.

Recommended filenames:
- clearstack-logo-horiz.svg      (header/logo lockup)
- clearstack-mark.svg            (square mark)
- favicon.ico, icon-192.png, icon-512.png, apple-touch-icon.png (optional)

Notes:
- Prefer SVG with transparent background.
- When updating assets, consider filename versioning (e.g., `...-v2.svg`) to bust caches.
